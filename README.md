# Adaptive Web Page: Click & Type

# Практична робота №3: Розробка адаптивної Web-сторінки

## 1. Мета проєкту

Створення сучасного, адаптивного інтерфейсу для інтернет-магазину механічних клавіатур з використанням семантичної верстки та підходу Mobile-first.

## 2. Семантична структура та HTML-розмітка

Під час розробки було віддано перевагу семантичним тегам HTML5 замість нескінченних контейнерів `<div>`. Це забезпечує правильну ієрархію документа та покращує доступність (Accessibility):

- `<header>` — містить навігацію, логотип та функціональні іконки.
- `<nav>` — відокремлює блок посилань для переміщення по сайту.
- `<main>` — контейнер для унікального контенту сторінки.
- `<section>` — використано для групування контенту за смислом: Hero-банер, фільтри форматів, популярні товари.
- `<article>` — застосовано для кожної картки товару як незалежного компонента.
- `<footer>` — містить авторські права та допоміжні посилання.

## 3. Технології побудови Layout (Макет)

### Використання CSS Grid

Для головного каталогу товарів (`.product-grid`) використано **CSS Grid**. Це дозволило створити дворівневу систему сітки, де кількість колонок змінюється залежно від ширини екрана за допомогою властивості `grid-template-columns: repeat(x, 1fr)`. Це найбільш зручний спосіб побудови складних адаптивних інтерфейсів.

### Використання Flexbox

Для елементів, що вимагають вирівнювання в одну лінію або центрирування, використано **Flexbox**:

- Розподіл простору в шапці сайту через `justify-content: space-between`.
- Вирівнювання контенту всередині карток товарів по вертикальній осі (`flex-direction: column`).
- Центрування кнопок вибору форматів клавіатур.

## 4. Адаптивність та Mobile-first підхід

Проєкт реалізовано за принципом **Mobile-first**: спочатку написані стилі для найменших екранів, а потім додані розширення через Media Queries.

### Точки перелому (Breakpoints):

1.  **Mobile (до 768px)**: Вертикальне відображення меню, прихована навігація під кнопку-бургер, компактна сітка товарів.
2.  **Tablet (768px – 1024px)**: Розширення шрифтів, збільшення відступів (`padding`), сітка товарів стає триколонковою.
3.  **Desktop (1024px+)**: Повна трансформація шапки. Меню стає видимим у рядку, додається складна анімація підкреслення посилань `::after` з розширенням від центру. Сітка товарів розширюється до 4 колонок.

---

## 5. Робота з графікою та шрифтами

- **SVG**: Всі іконки (пошук, кошик) вбудовані як векторні елементи. Це гарантує ідеальну чіткість на дисплеях та високу швидкість завантаження.
- **Box Model**: Для всіх елементів встановлено `box-sizing: border-box`, що дозволяє точно контролювати розміри з урахуванням `padding` та `border`.
- **Інтерактивність**: Для реалізації мобільного меню використано мінімальний JavaScript код, що перемикає клас `.active` для навігаційної панелі.
